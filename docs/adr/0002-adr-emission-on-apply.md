# ADR emission rules on Apply

Apply writes Matt Pocock–style ADRs to `docs/adr/NNNN-slug.md` only for **kept** assumptions that are hard to reverse, surprising without context, and real trade-offs. Kills land in `CHAR.md`; research remakes supersede prior ADRs; “That’s enough” skips create no ADRs. Always also write `CHAR.md` and constraint bullets into `AGENTS.md`.
